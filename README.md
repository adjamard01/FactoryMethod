# Design Pattern Factory Method

In deze opdracht werk je met het **Factory Method** pattern.

In `Main.java` moet je de juiste imports en stukken code te commenten of uncommenten.

De volgorde is:

1. Je krijgt eerst een **Voorbeeld** van de docent.
2. Daarna maak je zelf de **Opdracht**.
3. Na de opdracht kun je **VoorbeeldNaOpdracht** gebruiken als inspiratie voor een meer volwassen toepassing van het Factory Method design pattern.
